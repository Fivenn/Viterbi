## Installation

Use Cmake or the build.sh script as a generic short cut.
It is recommended to read all provided options using:

```sh
./build.sh --help
```

For a local install:

```sh
./build.sh
```

For system wide install (may need administrator privileges):

```sh
./build.sh --install
```
